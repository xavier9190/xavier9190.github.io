O Desafio

A Baterias Moura produz milhares de unidades diariamente. O sistema legado sofria com dados isolados e latência de 24 horas, tornando a detecção de defeitos em tempo real impossível. Precisávamos de um pipeline capaz de ingerir, limpar e servir insights em menos de 30 segundos.

LÓGICA_DE_TRANSFORMAÇÃO

A transformação principal utiliza BigQuery SQL e dbt (Data Build Tool) para orquestrar a transição do Bronze para o Silver. Implementamos lógica personalizada para lidar com dados de telemetria que chegam com atraso da linha de produção.

  • check_circle Carregamento incremental reduz custo computacional em 45%
  • check_circle Verificações automatizadas de qualidade de dados com Great Expectations
AUTOVOLT_PIPELINE.SQL 
WITH raw_telemetry AS (
  SELECT
    sensor_id,
    timestamp_trunc(event_time, SECOND) as sec_ts,
    avg(voltage) OVER (PARTITION BY sensor_id) as avg_v
  FROM `moura-production.bronze.iot_stream`
)
INSERT INTO `moura-production.silver.production_metrics`
SELECT * FROM raw_telemetry
WHERE avg_v > 12.6 -- Filtrar Outliers
